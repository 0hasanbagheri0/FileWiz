# FileWizard

ابزارهای پیشرفته برای مدیریت و پردازش فایل‌ها در پایتون.

## 🚀 ویژگی‌ها

- کپی، جابجایی و حذف فایل‌ها و پوشه‌ها
- فشرده‌سازی و خارج‌سازی فایل‌های ZIP
- پیدا کردن فایل‌های تکراری
- تغییر نام گروهی با الگو
- پاکسازی پوشه بر اساس پسوند یا حجم
- دریافت اطلاعات کامل فایل‌ها
- تشخیص فایل‌های مخفی

## 📦 نصب

```bash
pip install filewizard
=======
📁 ابزارهای پیشرفته برای مدیریت فایل‌ها و پوشه‌ها در پایتون
